Fixed Action Button

  • Result
  • Page
  • Bean

A Fixed Action Button it's positioned in the bottom righthand corner of the page and should contain a set of actions that will appear on hover. Use it to change the icon in the center.

Horizontal

You can also choose to display the items horizontaly. The items will show on the left of the button.

Colors

You can choose the color of the button and for every item. The colors of the items, if not specified, it's random.

Click to toogle

By default the items show on hover. You can force the user to click on the button open the FAB.
